# Unity Clock Application  
A business-focused clock app with Timer/Stopwatch features.  

## Requirements  
- Unity 2021.3.4f1 (LTS)  
- UniRx (Reactive Extensions)  
- VContainer (Dependency Injection)  

## Features  
1. Clock with timezone support  
2. Timer (Start/Stop/Pause + sound)  
3. Stopwatch (Lap times)  

## Development  
1. Clone the repo.  
2. Open in Unity 2021.3.4f1.  

## Future Considerations  
- iOS/iPad UI adaptability  
- VR integration (optional)
